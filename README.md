# solGuard-AI

## Overview
SolGuard AI is an AI-powered security platform built for the Solana ecosystem. It helps users identify scam tokens, phishing websites, rug pulls, and malicious wallets before interacting with them.

## Problem
Every day, thousands of new Solana tokens are launched. Many of them are scams, causing users to lose funds due to rug pulls, fake liquidity, and phishing attacks.

## Solution
SolGuard AI analyzes:
- Token liquidity
- Holder distribution
- Wallet behavior
- Smart contract risk
- AI-based scam probability

## Features
- Real-time Token Scanner
- Wallet Risk Score
- Rug Pull Detection
- Telegram Alerts
- Public API
- Web Dashboard
